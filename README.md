# Port-Scanner
A Python based TCP port scanner that identifies open ports on a target host.

FEATUREA
-Scan ports 1-1025.
-Timestamps each scan.
-Optimized timeout for fast results

USAGE
Run the script and enter a target IP address when prompted.

EXAMPLE OUTPUT
Scanning 127.0.0.1
Port 135 Open
Port 445 Open
Scan started at: 2026-05-07 16:32:01

TECHNOLOGIES 
-Python
-Socket Library
-Datetime Library
